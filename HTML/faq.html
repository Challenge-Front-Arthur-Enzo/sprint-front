Perguntas Frequentes (FAQ)

O que é o Lembre Saúde?

É um sistema que envia lembretes de consultas via WhatsApp para pacientes e cuidadores, ajudando a reduzir faltas.

Preciso baixar algum aplicativo?

Não. Todo o contato é feito por mensagens no WhatsApp, sem necessidade de instalar nada novo.

O sistema serve para agendar consultas?

Não. O agendamento continua sendo feito pelo sistema oficial do hospital. O Lembre Saúde apenas envia lembretes e confirmações.

Posso confirmar minha presença pela mensagem?

Sim. Basta responder "Sim" para confirmar ou "Não" para cancelar. O cuidador também recebe essas informações.

Existe uma interface para quem tem dificuldade no uso do celular/computador?

Sim. Existe uma tela com botões grandes, alto contraste e design acessível para facilitar o uso por idosos ou pessoas com dificuldades tecnológicas.
